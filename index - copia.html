Bienvenido/a a mi proyecto de JQuery UI

He preparado esto para que puedas probar todos los widgets, efectos e interacciones que nos ofrece esta librería.

Interacciones

Las interacciones agregan comportamientos básicos basados ​​en el mouse a cualquier elemento. Puede crear listas ordenables, elementos redimensionables, comportamientos de arrastrar y soltar y más con solo unas pocas líneas de código. Las interacciones también son excelentes bloques de construcción para aplicaciones y widgets más complejos.

Widgets

Los widgets son controles de interfaz de usuario con todas las funciones que llevan la riqueza de las aplicaciones de escritorio a la Web. Todos los widgets proporcionan un núcleo sólido con una gran cantidad de puntos de extensión para personalizar el comportamiento, así como compatibilidad con temas completos.

Efectos

Los efectos añaden soporte para animar colores y transiciones de clases, además de proporcionar varias facilidades adicionales. Además, se encuentra disponible un conjunto completo de efectos personalizados para usar al mostrar y ocultar elementos o simplemente para agregar algo de atractivo visual.
